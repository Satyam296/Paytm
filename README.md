## Getting Started

### Prerequisites

- Node.js v16+  
- MongoDB (local or cloud)  
- npm or yarn  

### Installation

```bash
# Clone the repo
git clone https://github.com/Satyam296/paytm-wallet-clone.git
cd paytm-wallet-clone

# Backend setup
cd backend
npm install
npm run dev

# Frontend setup (in a new terminal)
cd ../frontend
npm install
npm run dev
```
